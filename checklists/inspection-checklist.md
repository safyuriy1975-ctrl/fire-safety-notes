# Чек-лист осмотра объекта

Используется при плановом осмотре объекта пожарной безопасности.

## 1. Системы пожарной сигнализации (АПС)

- [ ] Прибор приёмно-контрольный (ППК) — горит индикатор «Норма»
- [ ] Нет сигналов «Неисправность», «Отключение»
- [ ] Пожарные извещатели — визуальный осмотр, нет повреждений
- [ ] Ручные пожарные извещатели (ИПР) — не вскрыты, на месте
- [ ] Журнал ТО подписан и актуален

## 2. Система оповещения (СОУЭ)

- [ ] Оповещатели (сирены, табло «Выход») — на месте, не закрыты
- [ ] Проверка звукового сигнала (тест)
- [ ] Световые указатели «Выход» — горят в дежурном режиме
- [ ] Аварийное освещение — работает при отключении питания

## 3. Огнетушители

- [ ] На месте согласно схеме расстановки
- [ ] Пломбы не нарушены
- [ ] Дата поверки актуальна (не просрочена)
- [ ] Свободный доступ (не загорожены)

## 4. Пути эвакуации

- [ ] Коридоры и выходы не захламлены
- [ ] Двери эвакуационных выходов открываются свободно
- [ ] Знаки эвакуации видны и освещены
- [ ] Запасные выходы не заперты на ключ изнутри

## 5. Видеонаблюдение (если установлено)

- [ ] Все камеры работают (нет «чёрных» экранов)
- [ ] Запись ведётся (проверить на регистраторе)
- [ ] Дата/время на регистраторе актуальны

## 6. Документация на объекте

- [ ] Инструкция о мерах ПБ — в наличии
- [ ] Схема эвакуации — вывешена на видном месте
- [ ] Приказ о назначении ответственного за ПБ — актуален
- [ ] Журнал инструктажей — заполнен
