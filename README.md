# Fire Safety Notes

Рабочий репозиторий по пожарной безопасности, обслуживанию объектов и техническим заметкам.

## Разделы
- [Нормативные материалы](./regulations/summary.md)
- [Чек-листы](./checklists/)
- [Шаблоны документов](./templates/)
- [Объекты](./objects/README.md)
- [Сервисные заметки](./service-notes/common-problems.md)

## Что сюда заносить
- выдержки из норм и правил (СП, ГОСТ, ФЗ-123);
- типовые замечания по объектам Печоры и региона;
- шаблоны актов, предписаний, отчётов;
- алгоритмы проверки систем АПС, видеонаблюдения;
- заметки по оборудованию, неисправностям, поставщикам;
- контакты клиентов и подрядчиков.

## Структура
```
fire-safety-notes/
├─ regulations/     — нормы и правила
├─ checklists/      — чек-листы проверок
├─ templates/       — шаблоны документов
├─ objects/         — карточки объектов
└─ service-notes/   — технические заметки
```
